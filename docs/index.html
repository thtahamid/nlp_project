A large commercial model (GPT‑5.2) labels Arabic handwritten images. A smaller open model (Qwen2.5‑VL‑7B) learns to replicate those labels through LoRA fine‑tuning — then runs locally at near‑zero cost.

// core concepts
Teacher · frozen
GPT-5.2 Vision
A large, expensive vision LM used only to generate training labels. Never shipped to production.
Student · trainable
Qwen2.5-VL-7B
Open 7B model. Small enough to self-host, capable enough to imitate the teacher after fine-tuning.
Adapter · r=32
Low-Rank Adaptation
Freeze the 7B. Inject tiny rank-32 matrices into attention and MLPs. Train <2% of weights.
Runtime · vLLM
OpenAI-compatible
One binary. LoRA swapped in at load. Same request shape as the OpenAI SDK — drop-in replacement.
