evaluate.py

Model Evaluation — CER · WER · Exact Match

1 CFG  config.py
BASE_MODEL, OUTPUT_DIR (lora adapter path)
MIN_PIXELS, MAX_PIXELS, EVAL_DATA path
2 BASE  Load Base Model
Qwen2_5_VLForConditionalGeneration
4-bit NF4 quantization (same as train)
device_map="auto"
2 PEFT  Load LoRA Adapter
PeftModel.from_pretrained(base, OUTPUT_DIR)
Merges adapter weights at load
model.eval() — disables dropout
3 PROC  AutoProcessor
from_pretrained(BASE_MODEL)
min_pixels / max_pixels from config
Used for tokenization + chat templates
4 DATA  Load Eval Dataset
Open config.EVAL_DATA (eval.jsonl)
json.loads per line → eval_samples[]
messages[-1]["content"] = expected output
5 TMPL  apply_chat_template
input_messages = messages[:-1] (no answer)
apply_chat_template(..., add_generation_prompt=True)
JSON prefix forcing: append '{"transcription":"'
5 SUPP  Token Suppression
Suppresses: tool_call, tool_response tokens
bad_words_ids built from tokenizer
Prevents model from switching tasks
6 GEN  model.generate()
max_new_tokens=512
temperature=0.1 | do_sample=False
bad_words_ids=bad_words_ids
Timed: records inference_time per sample
7 PARSE  parse_output()
1. Direct json.loads()
2. Strip markdown code blocks
3. Truncation search (scan backwards for })
4. Auto-complete suffixes
7 METRICS  Compute Metrics
CER: char error rate via difflib
WER: word error rate via difflib
exact_match: strip comparison
valid_json: parse_output not None
8 OUT  logs/eval_results.json
avg_cer | avg_wer
exact_match_rate (%) | valid_json_rate (%)
avg_inference_time (s/sample)
Per-sample log printed to stdout
