Image Pixel Bounds — Visual Reference

MIN_PIXELS · MAX_PIXELS · Qwen-VL patch convention

1 · What is a patch?

28×28 patch
Qwen-VL encodes images as 28×28 px patches — each patch = 1 visual token.
Pixel limits are expressed as multiples of 28² = 784 pixels.
